# セルフホスティングでブランディングを非表示にする

## なぜ Community Cloud では非表示にできないか

**Streamlit Community Cloud（無料）** では、以下の表示をアプリのコードから非表示にすることは**技術的にできません**。

- 「Created by ○○○」アイコン
- 「Hosted with Streamlit」バッジ
- Google 翻訳のプロンプトバー（ブラウザ側の制御）

これらはプラットフォームがレンダリングする要素のため、アプリ内の CSS / JavaScript からは変更できません。

## 解決策：セルフホスティング

**自分でサーバーを用意して Streamlit を動かす**ことで、上記のブランディングは表示されません。

### 方法1: Render（無料枠あり）

1. [Render](https://render.com) に登録
2. このリポジトリを GitHub に push
3. Render で New → Web Service
4. リポジトリを選択
5. 設定例:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `streamlit run app_simple.py --server.port=$PORT --server.address=0.0.0.0`
6. Deploy

※無料枠では15分の非アクティブでスリープし、起動に30秒ほどかかります。

### 方法2: Railway

1. [Railway](https://railway.app) に登録
2. GitHub リポジトリを接続
3. スタートコマンド: `streamlit run app_simple.py --server.port=$PORT --server.address=0.0.0.0`
4. デプロイ

### 方法3: Docker（VPS / 自前サーバー）

```bash
# ビルド
docker build -t aisatei-app .

# 実行
docker run -p 8501:8501 aisatei-app
```

## 翻訳プロンプトについて

Google 翻訳バーはブラウザ側の機能です。対策としては:

1. **Chrome**: アドレスバー右の翻訳アイコン → 「このサイトでは翻訳しない」を選択
2. **設定**: 設定 → 言語 → 「ページの言語と異なる場合に翻訳を提案する」をオフ
