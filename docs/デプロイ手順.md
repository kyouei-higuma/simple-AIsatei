# シンプル版査定システム デプロイ手順

Streamlit Community Cloud にデプロイし、ホームページからリンクできるURLを取得する手順です。

---

## 事前準備

### 必要なもの

- **GitHub アカウント**（未登録の場合は [github.com](https://github.com/) で作成）
- **Streamlit Community Cloud アカウント**（GitHub でログインするため、実質 GitHub があればOK）
- **Google Chat の Webhook URL**（査定結果の通知先。未設定でも査定は動作します）

---

## ステップ1: GitHub にリポジトリを作成・プッシュ

### 1-1. リポジトリの作成

1. [GitHub](https://github.com/) にログイン
2. 右上の **「+」** → **「New repository」** をクリック
3. 以下を設定：
   - **Repository name**: 例）`property-valuation-app`
   - **Public** を選択
   - **「Create repository」** をクリック

### 1-2. .gitignore の作成（重要）

`secrets.toml` を GitHub に上げないため、プロジェクト直下に `.gitignore` を作成します。

```
.streamlit/secrets.toml
__pycache__/
*.pyc
venv/
.env
```

### 1-3. プロジェクトを GitHub にプッシュ

プロジェクトフォルダで以下を実行（PowerShell または コマンドプロンプト）：

```powershell
cd "C:\Users\MD03\Desktop\不動産仮査定Project"

# Git が未初期化の場合
git init

# 全ファイルを追加（.gitignore で除外されたものは除く）
git add .

# 初回コミット
git commit -m "初回コミット: シンプル版査定アプリ"

# GitHub のリポジトリをリモートに追加（URLはあなたのリポジトリに置き換え）
git remote add origin https://github.com/あなたのユーザー名/property-valuation-app.git

# プッシュ
git branch -M main
git push -u origin main
```

> **注意**: `data/reins_data_3years.csv` は **必ず GitHub に含めてください**。アプリがこのファイルを参照して査定を行います。`.gitignore` に `data/` を入れないでください。

---

## ステップ2: Streamlit Community Cloud にデプロイ

### 2-1. Streamlit Cloud にアクセス

1. ブラウザで [https://share.streamlit.io/](https://share.streamlit.io/) を開く
2. **「Sign up」** または **「Log in」** をクリック
3. **「Continue with GitHub」** で GitHub アカウントと連携

### 2-2. アプリを作成

1. 右上の **「New app」** をクリック
2. 以下を入力：

   | 項目 | 入力内容 |
   |------|----------|
   | **Repository** | あなたの GitHub ユーザー名 / リポジトリ名（例: `username/property-valuation-app`） |
   | **Branch** | `main` |
   | **Main file path** | `app_simple.py` |

3. **「Advanced settings」** を展開（オプション）
   - **Python version**: 3.12（デフォルトのままでOK）

4. **「Deploy!」** をクリック

### 2-3. デプロイ完了を待つ

- 初回は **3〜5分程度** かかることがあります
- ログが表示され、完了すると **「Your app is live!」** と表示されます
- URL は `https://リポジトリ名-あなたのユーザー名-xxxxx.streamlit.app` の形式です

## ステップ3: Google Chat への通知を設定（任意）

査定結果を Google Chat に送りたい場合：

1. デプロイしたアプリの画面で、右上 **「⋮」**（三点メニュー）→ **「Settings」** をクリック
2. 左メニューから **「Secrets」** を選択
3. 次の形式で入力（引用符は半角 `"` を使用）：

   ```
   WEBHOOK_URL = "https://chat.googleapis.com/v1/spaces/XXXXX/messages?key=XXXXX&token=XXXXX"
   ```

4. **「Save」** をクリック
5. 30秒〜1分待ってから、アプリを再読み込みして動作確認

> 詳細は [Google_Chat_設定手順.md](Google_Chat_設定手順.md) を参照してください。

## ステップ4: ホームページにリンクを貼る

### 4-1. URL の取得

デプロイ完了後、Streamlit Cloud のアプリ画面に表示される URL をコピーします。

例: `https://property-valuation-app-username-xxxxx.streamlit.app`

### 4-2. ホームページへの貼り付け例

**HTML の場合：**

```html
<a href="https://あなたのアプリURL.streamlit.app" target="_blank" rel="noopener">
  無料AI査定はこちら
</a>
```

- `target="_blank"` … 新しいタブで開く
- `rel="noopener"` … セキュリティのため推奨

**ボタン風の例：**

```html
<a href="https://あなたのアプリURL.streamlit.app" 
   target="_blank" 
   rel="noopener"
   style="display:inline-block; padding:12px 24px; background:#3498db; color:white; text-decoration:none; border-radius:6px;">
   🏠 無料AI査定を試す
</a>
```

## トラブルシューティング

### デプロイが失敗する

- **requirements.txt** がリポジトリに含まれているか確認
- **app_simple.py** のパスが正しいか確認（`app_simple.py` がルートにあること）
- **data/reins_data_3years.csv** がリポジトリに含まれているか確認

### アプリが起動しない

- Streamlit Cloud の **Logs** でエラー内容を確認
- ローカルで `streamlit run app_simple.py` が動作するか確認

### 査定結果が Google Chat に届かない

- Streamlit Cloud の **Settings → Secrets** に `WEBHOOK_URL` が正しく設定されているか確認
- 引用符が半角 `"` であること（スマートクォート `"` は不可）
- Secrets 保存後、1分ほど待ってから再試行

2. `git push origin main` で GitHub にプッシュ
3. Streamlit Cloud が自動で再デプロイ（数分で反映）
